# covid-19

疫情数据统计，数据来源于 http://m.sh.bendibao.com/news/gelizhengce/fengxianmingdan.php

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Run test,

```bash
$ npm test
```

Build library via `father-build`,

```bash
$ npm run build
```
